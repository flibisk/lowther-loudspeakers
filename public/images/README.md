# Lowther Loudspeakers - Image Assets

This folder contains all the image assets for the Lowther Loudspeakers website.

## Folder Structure

```
images/
├── hero/                    # Hero section images
│   └── hero-home.jpg       # Main hero background
├── collections/            # Collection showcase images
│   ├── philharmonic.jpg    # Philharmonic collection
│   └── grand-opera.jpg     # Grand Opera collection
├── speakers/               # Individual speaker images
│   ├── philharmonic-iii-1.jpg
│   ├── grand-opera-1.jpg
│   ├── cottage-1.jpg
│   └── quarter-wave-1.jpg
├── blog/                   # Blog post images
│   ├── building-quarter-wave.jpg
│   └── choosing-amp.jpg
├── diy/                    # DIY/Build section images
│   └── diy-workshop.jpg
├── reviews/                # Customer review images
├── icons/                  # Icon files (SVG, PNG)
└── README.md               # This file
```

## Image Requirements

### Hero Images
- **Format**: JPG or WebP
- **Size**: 1920x1080px minimum
- **File size**: < 500KB for web optimization

### Speaker Images
- **Format**: JPG or WebP
- **Size**: 800x600px minimum
- **Aspect ratio**: 4:3 or 16:9
- **File size**: < 300KB each

### Collection Images
- **Format**: JPG or WebP
- **Size**: 1200x800px minimum
- **File size**: < 400KB each

### Blog Images
- **Format**: JPG or WebP
- **Size**: 800x450px (16:9 aspect ratio)
- **File size**: < 250KB each

## Naming Convention

- Use lowercase letters
- Use hyphens (-) instead of spaces
- Be descriptive: `philharmonic-iii-front-view.jpg`
- Include model names: `grand-opera-wood-finish.jpg`

## Optimization Tips

1. **Compress images** before uploading
2. **Use WebP format** for better compression
3. **Include alt text** in the data files
4. **Optimize for retina displays** (2x resolution)

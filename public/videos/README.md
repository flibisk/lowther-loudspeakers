# Lowther Loudspeakers - Video Assets

This folder contains all video content for the Lowther Loudspeakers website.

## Supported Formats

- **MP4** (recommended for web)
- **WebM** (for better compression)
- **MOV** (for high-quality content)

## Folder Structure

```
videos/
├── hero/                   # Hero section videos
├── product-demos/          # Product demonstration videos
├── testimonials/           # Customer testimonial videos
├── manufacturing/          # Behind-the-scenes content
├── tutorials/              # DIY and setup tutorials
└── README.md               # This file
```

## Video Requirements

### Hero Videos
- **Format**: MP4
- **Resolution**: 1920x1080px (1080p)
- **Duration**: 30-60 seconds
- **File size**: < 10MB for web optimization
- **Codec**: H.264

### Product Demos
- **Format**: MP4
- **Resolution**: 1280x720px minimum
- **Duration**: 2-5 minutes
- **File size**: < 50MB

### Testimonials
- **Format**: MP4
- **Resolution**: 1280x720px
- **Duration**: 30-90 seconds
- **File size**: < 20MB

## Optimization Tips

1. **Compress videos** for web delivery
2. **Include subtitles** for accessibility
3. **Add thumbnails** for better UX
4. **Use progressive download** for faster loading

## Naming Convention

- Use lowercase letters
- Use hyphens (-) instead of spaces
- Be descriptive: `philharmonic-iii-sound-demo.mp4`
- Include duration: `grand-opera-unboxing-2min.mp4`

## Usage in Website

Videos can be embedded using:
- HTML5 `<video>` tags
- Next.js video components
- External platforms (YouTube, Vimeo)

Drop your videos into the appropriate folders and they'll be available for use throughout the website!

# Answer Engine Optimization (AEO) Guide

## Overview

Answer Engine Optimization (AEO) prepares your website for AI-powered search engines like ChatGPT, Claude, Perplexity, and Google's AI Overview. This guide explains how the Lowther website is optimized for these platforms.

---

## What is AEO?

AEO differs from traditional SEO by focusing on:
- **Direct answers** rather than rankings
- **Structured data** that AI can easily parse
- **Clear semantic markup** for context
- **Natural language** that AI can understand and cite
- **Comprehensive information** in digestible formats

---

## Current AEO Implementation

### 1. **Structured Data (JSON-LD)**

We implement multiple schema types recognized by AI engines:

#### Organization Schema
Located in `/src/app/layout.tsx` - tells AI about your company:

```typescript
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Lowther Loudspeakers",
  "foundingDate": "1932",
  "address": {...},
  "contactPoint": {...},
  "sameAs": [...]
}
```

**Why it matters**: AI engines use this to answer questions like "Who makes Lowther speakers?" or "How old is Lowther?"

#### WebSite Schema with Sitelinks
Located in `/src/app/layout.tsx` - enables search functionality:

```typescript
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "{search_term_string}"
  }
}
```

**Why it matters**: Google displays sitelinks in search results, helping users find specific pages directly.

#### BreadcrumbList Schema
Auto-generated by `/src/components/breadcrumbs.tsx`:

```typescript
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [...]
}
```

**Why it matters**: Helps AI understand site structure and page hierarchy for better context in answers.

#### Product Schema
Use for loudspeakers, drive units, and products:

```typescript
const schema = generateStructuredData('Product', {
  name: 'Philharmonic III',
  description: '...',
  images: [...],
  price: '8995.00',
  currency: 'GBP',
  inStock: true,
});
```

**Why it matters**: AI can cite specific product details, prices, and availability.

#### Article Schema
Use for blog posts and editorial content:

```typescript
const schema = generateStructuredData('Article', {
  title: 'Building the Perfect Horn',
  description: '...',
  author: 'Lowther Team',
  publishedTime: '2024-01-15',
});
```

**Why it matters**: AI attributes quotes and information to your articles correctly.

## 3. **Metadata Optimization**

Every page should have comprehensive metadata via `generateSEOMetadata()`:

```typescript
import { generateSEOMetadata } from "@/lib/seo";
import { getPageMetadata } from "@/lib/metadata-config";

const pageMetadata = getPageMetadata("page-path");

export const metadata = generateSEOMetadata({
  title: pageMetadata.title,
  description: pageMetadata.description,
  keywords: pageMetadata.keywords,
  image: pageMetadata.ogImage,
  url: "/page-path",
});
```

### Metadata Components

1. **Title**: Clear, descriptive (50-60 characters)
2. **Description**: Comprehensive summary (150-160 characters)
3. **Keywords**: Relevant terms for context
4. **OG Image**: Visual preview (1200x630px)

## 4. **Content Optimization for AI**

### Writing for AI Engines

#### ‚úÖ DO:
- Use clear, descriptive headings
- Write in natural language
- Include specific facts and figures
- Define technical terms
- Use structured lists for processes
- Answer common questions directly

#### ‚ùå DON'T:
- Use vague or marketing-heavy language
- Hide information in images
- Use excessive jargon without explanation
- Create thin content pages
- Duplicate content across pages

### Example: Good AEO Content

```markdown
## What Makes Lowther Drive Units Special?

Lowther drive units are full-range speakers featuring:

1. **Single Driver Design**: No crossover networks needed
2. **High Efficiency**: 94-100dB sensitivity
3. **Handcrafted Quality**: Made in Britain since 1932

```

**Why it's good**: Direct answer, structured format, specific details, clear benefits.

## 5. **FAQ Schema (Optional)**

For pages with common questions, add FAQ schema:

```typescript
const faqSchema = generateStructuredData('FAQPage', {
  faqs: [
    {
      question: 'How long does it take to build a Quarter Wave?',
      answer: 'Most builders complete the Quarter Wave in 40-60 hours over 2-4 weeks...'
    },
    {
      question: 'What tools do I need?',
      answer: 'Basic woodworking tools including: table saw, router, drill press...'
    }
  ]
});
```

Add to page:
```tsx
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{ __html: JSON.stringify(faqSchema) }}
/>
```

## Common AEO Mistakes to Avoid

### 1. **Over-Optimization**
Don't stuff keywords unnaturally. AI detects and deprioritizes this.

### 2. **Thin Content**
Pages with <300 words are rarely cited by AI. Provide depth.

### 3. **Technical Errors**
Invalid structured data can hurt AI understanding. Validate regularly.

### 4. **Ignoring Mobile**
AI engines prioritize mobile-friendly content.

### 5. **Missing Context**
Don't assume AI knows your industry jargon. Define terms.
